# Portfolio assets

Drop your profile portrait here as **`profile.jpg`** (or `.png` — then update
`avatar` in `src/portfolio/data.ts`).

- Path the site loads: `/portfolio/profile.jpg`
- Recommended: square, ~512×512, < 300 KB.
- Until it's added, the hero avatar shows a styled "SN" monogram fallback.

The anime portrait you shared works great here — just save it as
`public/portfolio/profile.jpg`.
